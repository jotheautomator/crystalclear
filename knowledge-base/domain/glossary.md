# Glossary — CrystalClear

Linguagem ubíqua do domínio. Todos os agentes devem usar estes termos de forma consistente no código, testes e documentação.

| Termo (EN)              | Termo (PT)                    | Definição                                                                                               |
|-------------------------|-------------------------------|---------------------------------------------------------------------------------------------------------|
| Political Agent         | Agente Político               | Pessoa que exerce funções políticas: deputado, ministro, vereador, presidente de câmara/junta.          |
| Declaration of Interests| Declaração de Interesses      | Documento formal submetido por um agente político, declarando rendimentos, posições, ativos e participações. |
| Initial Declaration     | Declaração Inicial            | Submetida quando o agente inicia um mandato.                                                            |
| Regular Declaration     | Declaração Regular            | Submetida periodicamente (anualmente) durante o exercício de funções.                                   |
| Exceptional Declaration | Declaração Excecional         | Submetida quando há alteração significativa nos valores ou por pedido da Comissão de Ética.              |
| Institution             | Instituição                   | Entidade organizacional: empresa, partido, fundação, instituto, associação, organismo público.          |
| Position                | Cargo/Posição                 | Função exercida por um agente político numa instituição (pública, privada ou social).                   |
| Function                | Função                        | Tipo de função política ou profissional (deputado, consultor, administrador, etc.).                     |
| Asset                   | Ativo/Património              | Bem possuído pelo agente: imóvel urbano ou rural, com valor estimado.                                   |
| Business Participation  | Participação Empresarial      | Quotas, ações ou participações detidas pelo agente numa empresa.                                        |
| Support/Subsidy         | Apoio/Subsídio                | Valor recebido por um agente de uma instituição.                                                        |
| Ethics Committee        | Comissão de Ética             | Órgão responsável por validar declarações, verificar incompatibilidades e investigar violações.         |
| Complaint               | Queixa/Denúncia               | Comunicação de um cidadão sobre comportamento de um agente político que considera pouco transparente.   |
| Validation              | Validação                     | Processo pelo qual a Comissão de Ética aprova ou rejeita uma declaração.                                |
| Review (of declaration) | Parecer                       | Comentário da Comissão de Ética sobre uma secção/item de uma declaração.                                |
| Integrated Situation    | Situação Integrada            | Visão consolidada de todas as posições, ativos e participações de um agente numa data específica.       |
| Income Evolution        | Evolução de Rendimento        | Análise temporal dos rendimentos de um agente entre duas datas.                                         |
| Temporal Edge           | Relação Temporal              | Relação no grafo com datas de início e fim, permitindo queries por data.                                |
| Influence Network       | Rede de Influência            | Grafo de relações entre agentes, instituições e ativos.                                                 |
| Registration Request    | Pedido de Registo             | Solicitação de um utilizador para se registar na plataforma com um determinado role.                    |
| Role                    | Perfil/Papel                  | Tipo de utilizador: Admin, Political Agent, Citizen, Journalist, Ethics Committee.                      |
| Sensitive Data          | Dados Sensíveis               | Informação que deve ser parcialmente omitida consoante o role do utilizador que a consulta.              |
